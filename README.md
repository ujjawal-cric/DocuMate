# 📄 DocuMate

DocuMate is a  modern document converter and voice reader.  
It allows you to:

- ✅ Convert **PDF → Text**
- ✅ Convert **Word (DOCX) → Text**
- ✅ Convert **Word (DOCX) → PDF**
- 🔊 **Read text aloud** using browser's voice
